# Cluster / Worker Threads Experiment Example #

## Install
* Clone project:
```bash 
$ git clone <project-url>
```
* Install dependency package:
```bash
$ npm install
```

## Build and Run
### Cluster
* Run (use node):
```bash
$ node cluster.js
```
* Run (use npm):
```bash
$ npm run cluster
```

### Worker Threads
* Run (use node):
```bash
$ node --experimental-worker worker_threads.js
```
* Run (use npm):
```bash
$ npm run worker_threads
```
